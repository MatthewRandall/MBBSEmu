Command Line Options:
-?                  This Help Screen
-C <moduleConfig>   A Module Configuration File which allows for multiple modules to be
                    loaded and executed
-M <identifier>     Module Identifier (Overrides -C)
-P <path>           Path To Module (Overrides -C)
-APIREPORT          Generate API Report
                    This generates a JSON file which details the imported APIs used by the
                    module, assisting in development of MBBSEmu
-DBRESET <password> Reset the SQLite DB back to default (deletes everything), if no password
                    is specified, you will be prompted to enter one

Examples: 
./MBBSEmu -M MyModule -P c:\bbsv6\ -dbreset sysop
./MBBSEmu -C moduleConfig.json

appsettings.json Options:

BBS.Title               String         Title (name) of your Bulletin Board, used by MAJORBBS.H
GSBL.Activation         Number         Activation Code for MajorBBS & Worldgroup (8-Digit)
Module.DoLoginRoutine   True/False     Display Module Post-Login Messages (True/False)
Telnet.Port             Number         Port Number for Telnet to listn on (Default: 23)
ManagementAPI.Enabled   True/False     Enable/Disable the Management REST API (Default: False)
ManagementAPI.Port      Number         Port Number for the Management REST API (Default: 8080)
ManagementAPI.Secret    String         Secret Key used for generating JWT Authorization Tokens
                                       Minimum 16 characters, missing/default will generate a new
                                       GUID token on each startup
Database.File           String         Filename for the MBBSEmu SQLite Database
